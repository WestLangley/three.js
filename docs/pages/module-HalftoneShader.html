HalftoneShader

Import

HalftoneShader is an addon, and must be imported explicitly, see Installation#Addons.

import { HalftoneShader } from 'three/addons/shaders/HalftoneShader.js';

Properties

.HalftoneShader : ShaderMaterial~Shader (inner, constant)

RGB Halftone shader.

Used by HalftonePass.

Shape (1 = Dot, 2 = Ellipse, 3 = Line, 4 = Square) Blending Mode (1 = Linear, 2 = Multiply, 3 = Add, 4 = Lighter, 5 = Darker)
